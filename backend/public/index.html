API de Películas

Bienvenido a la documentación de la API de películas. Esta API proporciona acceso a información sobre películas y series.

Endpoints Colección Peliculas:

  • GET - /peliculas: Obtener todas las películas
  • GET - /peliculas/id: Obtener una película por ID
  • POST - /peliculas: Crear una nueva película
  • PUT - /peliculas/id: Actualizar una película existente
  • DELETE - /peliculas/id: Eliminar una película por ID

Endpoints Colección Usuarios:

  • GET - /usuarios: Obtener todos los usuarios
  • GET - /usuarios/id: Obtener un usuario por ID
  • POST - /usuarios: Crear un nuevo usuario
  • PUT - /usuarios/email: Actualizar un usuario existente por email
  • DELETE - /usuarios/email: Eliminar un usuario por email
  • GET - /usuarios/filterByNombre: Filtrar usuarios cuyos nombres comiencen desde la A hasta la Z
  • GET - /usuarios/searchByName: Buscar por nombre de usuario
